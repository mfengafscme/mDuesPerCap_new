Apply Update Rebate Edit Summary Help



The Apply Update Rebate Edit Summary screen provides the user with the ability to view a summary report showing the impact of applying the associated update file. This screen will provide the user the ability to view pre-processing statistical information on the Rebate update and give the user the ability to apply or reject the update.
Select for Update (check box) The system provides the user with the list of pending Affiliates. At least one must be chosen before initiating the Apply Update process.
Perform Update (button) The system completes the requested update for the chosen Affiliates.
Cancel (button) The system returns the user to the Apply Update screen.
Reject (button) The system aborts the requested update and prompts the user for comments as to why they are canceling.
View (link) The system displays the Exception Detail screen.
