Participation Group Add Outcome Help



The system provides the user the ability to add a new Participation Outcome code to an existing Participation Detail.

Name and Description are required fields.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Submit (button) The system performs validations and saves the changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Participation Group View screen.
