Organization Associate Detail Help



This screen provides the user the ability to view an Organization Associate’s information.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Edit (Button) The system allows the user to edit the Organization Associate Detail Information.
Return (Button) The system displays a list of all the Organization Associates for this Organization.
Maintain Location Association (Link) The system allows the user to choose an Organization Location for this Organization Associate.
Maintain Email Addresses (Link) The system allows the user to add, edit, and delete Email information for this Organization Associate.
View More Comments (Link) The system will provide the user access to another screen that will show all the comments for this Organization Associate, not just the most recent one that is on this screen.
