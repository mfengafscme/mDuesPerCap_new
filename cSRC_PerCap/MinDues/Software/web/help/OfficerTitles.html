Officer Titles Help



The Officer Titles screen provides the user with the ability to view an Affiliate's Officer Title information.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Remove (link) The system allows the user to remove one of the existing officer titles. The action is only available for officer titles where the '# with title' field has a value of 1.
Add (link) The system allows the user to add a new officer title to the constitutional titles list.
View More Comments (link) The system will provide the user access to another screen that will show all the comments for affiliate's officer titles.
