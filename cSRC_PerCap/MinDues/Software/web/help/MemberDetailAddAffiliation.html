Member Detail Add - Affiliation Help



This screen provides the user with the ability to create Member Affiliation between a Member or Person and an Affiliate. This screen can only be entered from the two paths of either the add action on the Member Detail screen or the add action from the Verify Person Results screen called from the Add Member path. The only editable fields are the Affiliate Identifier and New Comments on this screen. The remaining sections are read only.

The Affiliate Finder is used throughout the site as a simple Affiliate Identifier search. The user must enter the Type and either the Local, State, or Council fields before selecting the Finder link. Once the Finder link is clicked, the system returns the result of that search in a new browser window. When one of the results are selected in that window, the window closes and the selected Affiliate Identifier is returned to the original screen and populated in the associated Affiliate Identifier fields. Under the Affiliate View Data Utility(VDU), the affiliate finder will only display affiliates found within the Affiliate Hierarchy of the VDU user.

All required fields must be entered.

The Affiliate Identifier must exist in the system.

A member cannot be added to an Affiliate with an Affiliate Status of Deactivated or Pending Deactivation or Administrative/Legislative Council.

The Member Type can only be ‘Retiree’ and ‘Retiree Spouse’ if Affiliate Type is ‘R’ or ‘S’.

If the Affiliate Finder functionality is not used to build the fields of the Affiliate Identifier and duplicate Affiliates are found for the data entered, the system will display the Duplicate Affiliate Identifier Notifier Screen as part of the flow to allow the user to indicate the correct Affiliate. Once the choice is made, the flow continues with the requested action.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (Button) The system validates and saves new Member data.
Submit & Add Another (Button) The system validates and saves new Member data, and allows the user to enter another Member.
Reset (Button) The system clears any changes made to the screen returning it to its original content.
Cancel (Button) The system returns to the detail screen from which the Member Detail Add Affiliation screen was initiated
