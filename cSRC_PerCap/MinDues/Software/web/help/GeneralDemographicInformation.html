General Demographic Information Help





This screen provides General Demographic Information for the current person. This screen is primarily for viewing this information. The General Demographic - Edit screen provides the ability to edit this information. Additional children are added through this screen, while information on a spouse is added and edited through the General Demographic - Edit screen. Information on existing children can be updated through the General Demographic - Edit screen as well.

This screen contains general demographic information such as Birth Date, Deceased Date, Marital Status and Ethnic Origin.

In the current release of the AFSCME Enterprise Application, Spouse and Children are not recorded as Person records and consequently will not show up through a Person search.

This screen displays the Person Tab at the top and bottom of the screen. Actions (links) displayed in the tab will vary depending on the privileges held by each user.More information on the Person Tab is provided below.

The following Actions (links or buttons) may be available from this screen. The actions available to the user will depend on the privileges that the user has.
Delete (link) Delete an existing child
Add (link) Displays the General Demographic Information - Add Child screen
