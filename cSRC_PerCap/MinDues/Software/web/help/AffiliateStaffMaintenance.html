Affiliate Staff Maintenance Help



The Affiliate Staff Maintenance screen provides the user the ability to view and delete a Staff Person associated with an Affiliate.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

All fields with an asterisk * are required fields.

The following Actions (links or buttons) may be available from this screen.
View (link) The system displays the selected Staff Person's Detail Information.
Remove (links) The system removes the Staff Person from the current list.
Previous (link) The system displays the previous group of Staff. Sort order is maintained. Active only if list are greater than 25 and the first group of results is not being viewed.
Next (link) The system displays the next group of results. Sort order is maintained. Active only if list are greater than 25 and the last group of results is not being viewed.
Sort (link)
Name, Affiliate Staff Title, Point of Contact For, Location, Phone Number, and Email Address		 The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
Add New (button) The system allows the user to Add a new Staff Person.
Return (button) The system displays the Affiliate Detail Information.
