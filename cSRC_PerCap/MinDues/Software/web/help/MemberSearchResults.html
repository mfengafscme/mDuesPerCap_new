Member Search Results Help



This screen provides the user with the ability to view the results of a Basic Member Search or a Power Member Search and to navigate the results, sort the results, view the details of an individual result, or initiate a new search.

The Select column allows selection of a member and see detailed information.

The columns are dependant on the "Fields to Display" selections made in the Basic Member Search or Power Member Search screens. Due to the number and size of fields to display, the user may be required to scroll to the right to view all of the fields.

If Phone Number had been selected, the Home Phone Number associated with the Affiliate Relations Department will display. If Email Address had been selected, the Primary Email Address will display.

Initial sort order will be by Name (Last Name then First Name) ascending alphabetically, State ascending alphabetically, Council ascending numerically and Local ascending numerically.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

View: (Link) The system displays detailed information about the selected Member.
Previous (Link) The system displays the previous group of results. Sort order is maintained.
Next (Link) The system displays the next group of results. Sort order is maintained.
Sort By Field (Link) The system displays the results sorted by the column selected. If the results were already sorted by that column, the results will be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based on the column selected.
New Search (Button) The system displays Basic Member Search Screen to allow the user to elect the basic search for a Member.
Add New Member (Button) The system allows the user to Add a new Member
