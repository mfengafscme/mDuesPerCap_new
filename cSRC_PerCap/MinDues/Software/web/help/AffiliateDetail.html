Affiliate Detail Help



The Affiliate Detail screen provides the user with the ability to view data of an individual Affiliate and give the user the ability to perform actions on the Affiliate's information.

The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality. Fields with an asterisk * are required fields.

Maintain Location Information (link) The system allows the user to add, edit, and delete Location information for this Staff Person.
View More Comments (link) The system will provide the user access to another screen that will show all the comments for Affiliate.
