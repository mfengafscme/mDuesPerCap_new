Political/Legislative Information Edit Help



The Political/Legislative Information - Edit screen provides the user with the ability to update a Person's record for political or legislative information such as:
  • Political Objector (checkbox),
  • DO NOT CALL (checkbox)
  • PAC COntributor (checkbox)

Please note that based on business rules in effect for the user, these fields may or may not be editable.

The rest of the fields on the page are not editable and are populated from external sources.

In addition to the checkboxes, the following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Political/Legislative Information screen without saving changes.
Submit (button) The system validates and saves the inputs made via the checkboxes described above.
