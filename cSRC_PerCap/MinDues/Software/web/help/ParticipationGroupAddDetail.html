Participation Group Add Detail Help



The Participation Group Add Detail screen provides the user the ability to add a new Participation Detail code to an existing Participation Type.

Name and Description are required fields.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (button) The system performs validations, saves the Detail.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Participation Group View screen without saving the new Detail.
