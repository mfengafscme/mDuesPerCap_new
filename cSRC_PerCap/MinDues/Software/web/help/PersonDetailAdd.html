Person Detail Add Help



This screen provides the user the ability to add a new Person.

Data that was entered on the Verify Person screen will be prepopulated in their respective fields on this screen.

If the Person being added has a value entered for Social Security Number, the system will check if that Social Security Number already exists. If it does already exist, the system will show the Duplicate SSN Notifier screen before performing the requested action, thus allowing the user to continue or cancel the action. If the user elects to continue the action, the records that match are flagged as duplicates.

If one of the Address fields is populated, the following rules will be validated:

  • If Country is Canada, the City and Province are required.
  • If one of the Phone fields is populated, then all of the remaining Phone fields are required.



    • The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality for the user.
    Submit (button) The system validates and saves new person data.
    Submit & Add Another (button) The system validates and saves new Person data and allows the user to enter another Person.
    Reset (button) The system clears any changes made to the screen returning it to its original content.
    Cancel (button) The system returns to the Main Menu.
