Apply Update Member Exception Detail Help



The Apply Update Member Exception Detail screen provides the user with the ability to view detail statistical information on an individual exception listed on either the Member Edit Summary or the Member Review Summary. This screen highlights the differences between the data received in the file and the data in the system for an individual.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

Return (button) The system returns the user to the Apply Update Member Review Summary Screen.
