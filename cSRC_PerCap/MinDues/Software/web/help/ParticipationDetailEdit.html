Participation Detail Edit Help



This screen provides the user the ability to edit a single Participation Detail record for an individual.

If a Shortcut value is entered, upon leaving the field, the screen will refresh and the Group, Type and Detail dropdowns will be populated and initialized to the values that correspond to that Shortcut.

If a value is changed in the Group dropdown, the Type dropdown will populate with the appropriate Type values for that Group. The Detail and Outcome dropdowns will be empty.

If a value is changed in the Type dropdown, the Detail dropdown will populate with the appropriate Detail values for that Group/Type combination. The Outcome dropdown will be empty.

If a value is changed in the Detail dropdown, the Outcome dropdown will populate with the appropriate Outcome values for that Group/Type/Detail combination.


The following Actions (buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Submit The system validates and saves the changes.
Reset The system clears any changes made to the screen returning it to its original content.
Cancel The system returns to Participation Detail Screen without any changes being saved.
