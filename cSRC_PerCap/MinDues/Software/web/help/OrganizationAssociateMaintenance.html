Organization Associate Maintenance Help



This screen provides the user the ability to view and delete an Organization Associate associated with an Organization.

Initial sort order will be by Name (Last Name then First Name) ascending alphabetically.

The following Actions (buttons or links) may be available from this screen. Remember that the actions that are available will depend on the user's privileges.
View (Link) The system displays the selected Organization Associate’s Detail Information.
Remove (Link) The system removes the Organization Associate from the current list.
Previous (Link) The system displays the previous group of Staff. Sort order is maintained.
Next (Link) The system displays the next group of results. Sort order is maintained.
Sort (Links) The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
Add New (Button) The system allows the user to Add a new Organization Associate.
Return (Button) The system displays the Organization Detail Information.
