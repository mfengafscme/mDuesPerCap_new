Location Maintenance Edit Help



The Location Maintenance Edit screen allows the user to modify data pertaining to locations associated with an organization. A location may have both a regular and a ship to address. Each location may have a both an Office and Fax phone number associated with it. This screen allows the location to be marked as primary - only one location of an organization may be marked as the primary location for that organization.

Available actions on this screen are based on the user's privileges. Key actions available on this page include:
Submit The system performs validations and saves changes.
Reset The system clears any changes made to the screen and returning it to its original content.
Cancel The system returns to Location Maintenance screen without saving any changes.
