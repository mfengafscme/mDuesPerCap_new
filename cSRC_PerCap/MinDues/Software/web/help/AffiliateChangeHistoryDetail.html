Affiliate Change History Detail Help



The Affiliate Change History Detail screen provides the user with the ability to view a subset of an Affiliate's history of changes from the results returned by the search on the Affiliate Change History screen.

Due to the number and size of fields to display, the user may be required to scroll to the right to view all of the fields.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.
The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.
Return (button) The system returns the user to the Change History search results.
New Search (button) The system allows the user to enter new search criteria for the Affiliate's Change History. The screen will have the last search results cleared from the screen.
