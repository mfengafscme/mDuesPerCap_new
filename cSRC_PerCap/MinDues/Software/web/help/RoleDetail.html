Role Detail Help



The system allows the user to edit a role name (text box) and description (text box). It also provides links to edit the reports, fields, and privileges associated with the role. Also serves as the Add Role screen.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Name (textbox)
Description (textbox)
Submit (button) The system performs validations and saves the Role Detail.
Delete (Button) Deletes this role.
Select Privileges (button) Provides the user with the Select Privileges screen.
Select Report Privileges (button) Provides the user with the Select Report Privileges screen.
Select Field Privileges (button) Provides the user with the Select Field Privileges screen.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Role Listing Screen.
