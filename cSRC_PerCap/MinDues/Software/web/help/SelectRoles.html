Select Roles Help



The system allows the user to view, add (check box), and edit (check box) user roles.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system performs validations and saves the selected roles.
Reset (button) The system clears any changes made to the screen and returning it to its original content.
Cancel (button) The system returns to User Edit screen.
