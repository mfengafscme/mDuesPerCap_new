Affiliate Search Results Help



The Affiliate Search Results screen provides the user with the ability to view the results of a Basic Affiliate Search or a Power Affiliate Search and gives the user the ability to navigate the results, sort the results, view the details of an individual result, or initiate a new search.

The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.

Sort fields are:
  • Affiliate Name
  • Affiliate Type
  • Local/Sub Chapter
  • State/National Type
  • Sub Unit
  • Council/Retiree Chapter

The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality.

View (links) The system displays information about the selected Affiliate.
Previous (link) The system displays the previous group of results. Sort order is maintained. This link is active only if results are greater than 25 and the first group of results is not being viewed.
Next (link) The system displays the next group of results. Sort order is maintained. This link is active only if results are greater than 25 and the last group of results is not being viewed.
Sort (links) The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
New Search (button) The system displays Basic Affiliate Search Screen to allow users to elect the basic search for an Affiliate.
