Affiliate Change History Help



The Affiliate Change History screen provides the user with the ability to search for an Affiliate's history of changes. When the user first comes to this screen, only the Search section is displayed. The Results section appears only after the user submits search criteria. The Search section remains along with the Results section to allow for additional searches to be performed.

  • Section is a list of the Affiliate areas that the system maintains history information.
  • From Date allows the user to identify the beginning range for a search in the history logs.
  • To Date allows the user to identify the end range for a search in the history logs.
To perform any action, the user must have the required privilege, which is set by the Admin functionality. The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.
Search (button) The system allows the user to search the Affiliate's Change History. At least on searchable field must have a value.
Reset (button) The system clears any values entered in the search fields.
Sort (links) The system displays the results sorted by the field the user selected (Section or Changed Date). If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
View (links) The system displays all the changes made for the given section by the user shown on the date listed.
