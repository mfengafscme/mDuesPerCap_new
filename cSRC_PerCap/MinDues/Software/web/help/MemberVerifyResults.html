Person Verify Results Help



This screen displays information on similar persons to help identify if the person to add already exists in the system. If so, the user can decide whether to continue to add the Member as a new Person, add the member as one of the existing Persons displayed, or choose a person to view for more detail information.

The following Actions (buttons or links) are available from this screen depending on the user's privileges set by the Admin functionality.
Previous (link) The system displays the previous group of results. Sort order is maintained.
Next (link) The system displays the next group of results. Sort order is maintained.
Sort (links) Sort fields are: Affiliate Type, Local, State/National Type, Sub Unit, Council, Concatenated Name, Address, City, State, Zip/Postal Code and SSN. The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
Add (links) The system continues on to the appropriate Add screen allowing the user to enter the Member or Staff person from the existing Person found.
View (links) The system displays detailed information about the selected person/member.
New Search (button) The system displays Verify Person screen allowing the user to verify another person.
Add New (button) The system continues on to the appropriate Member Detail Add screen allowing the user to enter this person if not found in the system.
