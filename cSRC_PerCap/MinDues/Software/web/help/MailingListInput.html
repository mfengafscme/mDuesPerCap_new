Person Mailing List by Mail Code Help



This screen provides information on how to use the Person Mailing List by Mail Code screen.

When the mailing list has been completed, an email notification will be sent containing the following information:
  • Number on the Mailing List (number of members/officers assigned to the mailing list).
  • Number created on the Mailing List (actual number of valid and mailable members/officer added to the file).
  • Number of records evaluated (number of members/officers that belong to a mailing list).
The following rule must be enforced when generating this report:
  • For the Name field on the report, use the Alternate Mailing Name field from the Person table if it contains a value. Otherwise, use a concatenation of the First Name, Middle Initial, Last Name, and Suffix fields.
  • If Deceased Flag is true, the Member cannot receive mail.
The following Action (button) are available from this screen depending on the user's privileges.

Submit The system generates a list of names and addresses of all persons that are on the mailing list of selected mail code.
