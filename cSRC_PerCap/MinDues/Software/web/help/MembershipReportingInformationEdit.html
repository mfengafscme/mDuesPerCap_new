Membership Reporting Information Edit Help



The Membership Reporting Information Edit screen provides the user with the ability to edit an Affiliate's Membership Reporting Information. Changes saved on this screen will apply to the Affiliate only and not be applied to the Sub-Affiliates. Application of changes to the Sub-Affiliates is done from the Enable Mass Change screen.

Note that for the Affiliate Status Change to option, the capability change the status to 'Merged', 'Deactivated', or 'Split' is only available via Enable Mass Change.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system validates and saves Membership Reporting Information changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to Membership Reporting Information Screen with no saved changes.
