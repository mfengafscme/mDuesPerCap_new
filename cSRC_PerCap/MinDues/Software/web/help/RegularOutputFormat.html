Select Output Format Help



This screen provides the user the ability to select output format from a list of formats.

The fields displayed on this screen vary based on the privileges set for the user.

Radio buttons for specifying the format of the extracting report are only enabled if one of the Extract file formats is selected.

Printed on Paper will produce a PDF report.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Generate Report (Button) The system provides the user the ability to generate the report in a selected format.
Cancel (Button) The system provides the user the ability to cancel the report generation process.
