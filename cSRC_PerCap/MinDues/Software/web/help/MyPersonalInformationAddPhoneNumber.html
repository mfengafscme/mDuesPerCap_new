My Personal Information Add Phone Number Help



The My Personal Information Add Phone Number screen provides the user with the ability to add a phone number associated with an individual.

By selecting the checkbox, the user can identify that the phone number should be considered their "Primary" phone number by checking the putting a check in the box marked "primary".

Required fields are type and number.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system performs validations and saves the new phone number.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the My Personal Information Screen.
