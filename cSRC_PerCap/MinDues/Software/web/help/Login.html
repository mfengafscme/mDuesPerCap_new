Login Help



The Login screen allows the user to enter user ID and password to access to the Enterprise Application.

Enter user ID (text box) The user account name.
Enter password (text box) The password for the user account.


The following Actions (buttons) are available from this screen.

Login The system validates user ID/password and grants the user access to the Enterprise Application.
Request Password The system allows the user to request a password to be emailed to them.


Information on the Header (Need Help with this page) and Footer links (AFSCME, PRIVACY POLICY, HELP) can be found by below:

AFSCME A link to www.afscme.org, AFSCME's web site
PRIVACY POLICY Displays a screen with information on AFSCME's privacy policy
HELP Provides the ability to download the End User Documentation for the AFSCME Enterprise Application system.
