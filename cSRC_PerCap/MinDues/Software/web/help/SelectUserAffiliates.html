Select Affiliate Privileges Help



The Select Affiliate Privileges screen provides the user with the ability to view the results of a Select Affiliate Privileges Search and gives the user the ability to navigate the results, sort the results, select the Affiliate Privileges, or initiate a new search.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system validates and saves the selected Affiliate Privileges and returns to User Edit screen.
Select All Results (button) The system checks all Affiliate Privileges on the selected checkbox.
Deselect All Results (button) The system unchecks all Affiliate Privileges from the selected checkbox.
New Search (button) The system displays Select Affiliate Privileges Search screen to allow users to search for another Affiliate.
Cancel (button) The system returns to User Edit screen.
Previous (link) The system displays the previous group of results. Sort order is maintained. This link is active only if results are greater than 25 and the first group of results is not being viewed.
Next (link) The system displays the next group of results. Sort order is maintained. This link is active only if results are greater than 25 and the last group of results is not being viewed.
Sort (links) Sort fields are: Affiliate Name, Affiliate Type, Local, State/National Type, Sub Unit, and Council/Retiree Chapter. The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
