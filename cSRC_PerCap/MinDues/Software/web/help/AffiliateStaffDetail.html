Affiliate Staff Detail Help



The Affiliate Staff Detail screen provides the user the ability to view an Affiliate Staff Person's information.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.

Edit (button) The system allows the user to edit the Staff Person Detail Information.
Return (button) The system displays a list of all the Staff Persons for this Affiliate.
Add (link) The system allows the user to add a Sub-Affiliate to this Staff Person's Locals Serviced list. Not available if the Staff Person works for a Retiree Sub Chapter or a Sub-Local.
Delete (links) The system deletes the Sub-Affiliate from this Staff Person's Locals Serviced list. The system prompts the user for confirmation of the requested action. Not available if the Staff Person works for a Retiree Sub Chapter or a Sub-Local.
Maintain Location Association (link) The system allows the user to choose an Affiliate Location for this Staff Person.
Maintain Email Addresses (link) The system allows the user to add, edit, and delete Email information for this Staff Person.
View More Comments (link) The system will provide the user access to another screen that will show all the comments for this Staff Person, not just the most recent one that is on this screen.
