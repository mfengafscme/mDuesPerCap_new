Mailing Lists Information Add Help



The Mailing Lists Information Add screen lets the user associate a Person or Organization to a specific mailing list. If the add is successful, the new mailing list will show up in the Mailing Lists Information screen for that Person or Organization. For a Person, select the mailing list. For an Organization, select the mailing list, and if relevant, add a number in the bulk count field for that Organization on that mailing list. Bulk count is used to identify the number of items to include in a bulk shipment. Bulk counts are infrequently used.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (button) Submits the request to add the person or organization to the Mailing List.
Reset (button) Resets the content of the screen to the values when the screen was first entered.
