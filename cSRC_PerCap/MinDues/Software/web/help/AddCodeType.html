Add Code Type Help



The system allows the user to Add a new Code Type by completing the Name (text box), Key (text box), Description (text box) and Category (pull-down).

The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality for the user.
Submit (button) The system performs validations and saves the new Code information.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Code Type Edit Screen.
