Select Report Privileges Help



The system allows the user to view, add (check box) and edit (check box) report privileges for this role.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system performs validations and saves the Select Report Privileges.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Role Detail Screen.


Note: Selecting a field name will allow the user to select or deselect all report privileges.
