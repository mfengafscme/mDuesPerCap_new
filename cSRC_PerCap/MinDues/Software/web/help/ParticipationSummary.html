Participation Summary Help



This screen provides the user the ability to view a summary of an individual’s Participation history.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

View (link) The system allows the user to view the Participation Detail Information.
Delete (link) The system deletes the Participation Detail Information.
Add (link) The system allows the user to add a new Participation Detail.
Sort (links) Group, Type, Detail, Outcome, and Date. The system redisplays the same screen with the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
