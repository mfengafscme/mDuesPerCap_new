Email Address Maintenance Help



The Email Address Maintenance screen provides the user with the ability to view, designate as "bad" or delete an email address associated with an individual.

The screen displays both primary and alternate email addresses (if known).

The user can designate the email address as "bad" to indicate that the email address is incorrect.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Edit (button) The system will display to the Email Address Maintenance - Edit screen for modifying email addresses.
Delete (link) The "Delete" link (next to each email address) will cause the system to delete the email address.
Return (button) The system returns to the detail screen from which the Email Address Maintenance Functionality was initiated.
