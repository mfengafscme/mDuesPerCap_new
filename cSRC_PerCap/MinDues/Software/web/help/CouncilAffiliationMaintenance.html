Council Affiliation Maintenance Help



The Council Affiliation Maintenance screen provides the user the ability to add a new Council Affiliation to the Affiliate.

All fields with an asterisk * are required fields.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Type (pull-down) The user selects the affiliate type C for Council or R for Retiree.
State/National Type (pull-down) The user selects the State/National type.
Council/Retiree Chapter (textbox) The user provides the Council Number.
Select (text with Affiliate Finder link).
Submit (button) The system validates and saves the Council Affiliation changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Charter Information Screen without saving any changes.
