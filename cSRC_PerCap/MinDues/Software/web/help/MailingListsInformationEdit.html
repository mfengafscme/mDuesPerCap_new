Mailing Lists Information Edit Help



This page provides the user the ability to edit the Bulk Count for a Bulk Mailing List.

The following Actions (buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Submit The system validates and saves the changes.
Reset The system clears any changes made to the screen returning it to its original content.
Cancel The system returns to the Mailing Lists Screen without saving the changes.
