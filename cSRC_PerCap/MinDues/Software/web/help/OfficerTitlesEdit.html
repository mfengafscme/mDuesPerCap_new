Officer Titles Edit Help



The Officer Titles Edit screen provides the user with the ability to edit an Affiliate's Officer Title information.

Automatic Delegate Provision (checkbox) The user is to select whether or not automatic delegate provision should be provided.
Affiliate Title (textbox) The user is to provide an affiliate title.
# with Title (textbox) The user is to provide number of offices for this title.
Month of Election (pull-down) The user is to select the month of election. If a position's Month of Election changes, the Initial Term End Date must be changed manually to reflect the new term.
Length of Term (pull-down) The user is to select the length of term. If a position's Length of Term changes, the Initial Term End Date must be changed manually to reflect the new term. Only the following positions are allowed to have a Length of Term equal to ‘Indefinite’:
  • Financial Reporting Officer
  • Administrator
  • Steward
  • Executive Director
  • Director
    		•
    Delegate Priority (numeric) The user is to provide delegate priority.
    RO (radio buttons) The user is to select whether or not to this officer title is a reporting officer.
    E-Bd (checkbox) The user is to select the whether or not this officer title is an executive board member.
    Affiliate Title (pull-down) The user is to select the affiliate title.
    Sub-Affiliate Title (pull-down) The user is to select the sub-affiliate title.
    Enter New Comments (textbox) The user is to provide comments.


    The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

    Submit (button) The system validates and saves Officer Title changes.
    Reset (button) The system clears any changes made to the screen returning it to its original content.
    Cancel (button) The system returns to Officer Title Screen.
