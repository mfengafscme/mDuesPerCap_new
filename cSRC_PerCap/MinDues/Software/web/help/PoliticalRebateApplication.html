Political Rebate - Application Help



The Political Rebate - Application screen lets the user view the details associated with an individual’s Rebate Application.

The following Actions (links) may be available from this screen. Remember that the actions that are available will depend on the user's privileges.

Return (Button) The system returns to the Political Rebate Summary By Year Screen.
Edit (Button) The system allows the Political Rebate Application to be edited.
Once the Application has been included in the Preliminary Roster and Rebate Update File generation, it can no longer be edited.
Clear (Link) The system will remove the Affiliate from the Application.
