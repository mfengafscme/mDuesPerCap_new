Affiliate Officer Maintenance Edit Help



The Affiliate Officer Maintenance Edit screen provides the user with the ability to edit, renew, remove, or replace an Affiliate's Officers.

Renew Available only when the End Term Year is the current year, and the office is filled.
Vacate Available only when the office is currently filled.
Replace Available both when the office is filled, to replace the current officer with a new one, and when the office is empty, to assign an officer.
End Term Year Only editable if the Replace option is chosen above.
First First name is only editable if the Replace option is chosen above.
Middle Middle name is only editable if the Replace option is chosen above.
Last Last name is only editable if the Replace option is chosen above.
Suffix Suffix is only editable if the Replace option is chosen above.
Stwd Steward Flag


If a name is entered and multiple results are found upon searching, a link is provided to view the search results and make a selection. If the name is changed in any way after returning from the search results, the search must be performed again.

If multiple office updates are made and submitted at once, all renew and vacate transactions, as well as all replace transactions that return one search result will be processed. A replace transaction that returns multiple results will not stop the processing. A message will be shown indicating that all transactions have been processed except for those showing multiple result messages.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.

Submit (button) The system performs validations and saves request changes. If the Name entered for an Officer being replaced yields more than one result, the system will return the user to the Affiliate Officer Maintenance Edit Screen allowing them to view the results for that position and choose the appropriate result for the replace.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Affiliate Officer Maintenance Screen without saving any changes.
Reset (links)
Next to each Person in the Potential Officers List		 The system clears the Select For Change Type, End Term, and Name fields associated with this action.
Click Here (links) The system displays a list of Officer Search Results allowing the user to choose the Person to use for the Replace functionality. Only viewable if the system finds multiple matches for a Name entered during a replace.
