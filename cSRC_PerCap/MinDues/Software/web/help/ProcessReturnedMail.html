Process Returned Mail Help



The Process Returned Mail screen provides the user with the ability to import a batch of bad addresses received as the result of a mailing.

To perform any action the user must have the required privilege set by the Admin functionality.

The following Actions (links or buttons) may be available from this screen.

Address Ids (text box) The system requires the user to provide address ids. To enter the address manually, the user can type the address ids and press the Enter key after each address id. The address ids can also be entered using a bar code scanner. This is a required field.
Submit (button) The system processes the returned mail, marks the bad address flag on the ones it can find, and displays a report summarizing what has and has not been processed.
Search (button) The system displays the Member search screen allowing the user to find the Member and mark the address as bad using the Maintain Addresses functionality.
Cancel (button) The system returns to the main menu without processing any returned mail.
