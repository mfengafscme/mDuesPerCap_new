Initiate NCOA Processing Help



The system allows the user to indicate that the NCOA data has been loaded into the appropriate tables using an external process and is ready for updating.

The NCOA Process has been scheduled response is provided to the user.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

OK (button) The system schedules the NCOA Process and returns the user to the Enterprise Application Main Menu.
X (button) The system does not schedule the NCOA Process and returns the use to the Enterprise Application Main Menu.
