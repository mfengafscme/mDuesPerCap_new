Affiliate Detail Add Help



The Affiliate Detail Add screen provides the user with the ability to add a new Affiliate.

The Fields that can add are:
Fields with an asterisk * are required fields.

Type (pull-down) The user is to provide an affiliate type.
Local/SubChapter (textbox) The user is to provide a local number.
State/National Type (pull-down) The user is to select a state or national type.
Sub Unit (textbox) The user is to provide a sub local code.
Council/Retiree Chapter (textbox) The user is to provide a council number.
Abbreviated Affiliate Name (textbox) The user is to provide an abbreviated affiliate name.
Affiliate Status (pull-down) The user is to select an affiliate status based on valid code values in database.
Sub-Locals Allowed (checkbox) The user is to select whether or not sub-locals are allowed.
AFSCME Legislative District (pull-down) The user is to select an AFSCME legislative district based on valid code values in database.
AFSCME Region (pull-down) The user is to select an AFSCME region based on valid code values in database.
Multiple Employers (checkbox) The user is to select whether or not multiple employers are allowed.
Annual Card Run Type (pull-down) The user is to select annual card run type based on valid code values in database.
Charter Name (textbox) The user is to provide a charter name. A charter name is not available for sub-locals and is not required if affiliate status is Pending or Not Chartered.
Charter Jurisdiction (textbox) The user is to provide a charter jurisdiction. A charter jurisdiction is not available for sub-locals and is not required if affiliate status is Pending or Not Chartered.
Charter Code (pull-down) The user is to select a charter code based on valid code values in database. A charter code is not available for sub-locals and is not required if affiliate status is Pending or Not Chartered.
Charter Date (pull-down and textbox) The user is to provide a charter date. A charter date is not available for sub-locals and is not required if affiliate status is Pending or Not Chartered.
County (textbox) The user is to provide a county. At least one county required. A county is not available for sub-locals and is not required if affiliate status is Pending or Not Chartered.
Approved Constitution (checkbox) The user is to select whether or not the constitution is approved.
Generate Default Officer Titles (checkbox) The user is to select whether or not the system is to generate default officer titles.
Affiliation Agreement Date (pull-down and textbox) The user is to provide an affiliation agreement date.


The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality.

Submit (button) The system validates and saves new Affiliate data.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the AFSCME Main Menu.
