Preliminary Roster, Rebate Update File Affiliates Selection Help



This screen provides the user the ability to view a list of Affiliates that have Preliminary Roster and/or Rebate Update File records, and allows the user to select the Affiliate(s) from the list. Once the Affiliate is selected and submitted, a Preliminary Roster report and a Rebate Update File report will be generated. These reports contain a listing of individuals that have requested Political Rebates for Membership within the selected Affiliates.

The following Actions (buttons or links) are available from this screen depending on the user's privileges.

Sort (links) Sort fields are: Type, Local/Sub Chapter, State/National Type, Sub Unit, and Council/Retiree Chapter. The system redisplays the same screen with the list sorted by the field the user selected. If the list was already sorted by the field the user selected, it should be displayed in the reverse order. Otherwise, the list is displayed as sorted in ascending order based upon the field the user selected.
Submit (button) The system generates the Preliminary Roster and the Rebate Update File reports.
Reset (button) The system clears any changes made to the screen and returning it to its original content.
Cancel (button) The system returns to Generate Reports Main Menu Screen.
