Apply Update Process Log Help



The Apply Update Process Log screen provides the user with the ability to search the log of processed reports.

The user can search processed reports by:
  • Date Range
  • Affiliate Type (pull-down)
  • Loc/Sub Chap (text box)
  • State/Nat’l (pull-down)
  • Sub Unit (text box)
  • CH/Ret Chap (text box)
  • Update Type (pull-down)
Processed (link) The processed link in the search results will provide the user with the Apply Update Review Summary screen.
Return (button) The system returns the user to the Apply Update screen without making changes.
Submit (button) The system performs validations and provides the user with the processed search results.
Reset (button) The system clears any changes made to the screen returning it to its original content.
