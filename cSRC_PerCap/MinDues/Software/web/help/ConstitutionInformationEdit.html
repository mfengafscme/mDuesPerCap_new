Constitution Information Edit Help



The Constitution Information Edit screen provides the user with the ability to edit an Affiliate's Constitution information.

Fields with an asterisk * are required fields.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Most Current Approval Date (textbox with date picker) The user is to provide a most current approval date.
Affiliation Agreement Date (textbox with date picker) The user is to provide an affiliation agreement date.
Method of Officer Election (pull-down) The user is to select a method of officer election based on valid code values in database.
Constitutional Regions (checkbox) The user is to select whether or not constitutional regions are to be included.
Meeting Frequency for Affiliate Meetings (pull-down) The user is to select meeting frequency for affiliate meetings based on valid code values in database.
Browse (text and button) The system allows the user to browse for a file to upload as the constitutional document.
Submit (button) The system validates and saves changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to Constitution Information Screen.
