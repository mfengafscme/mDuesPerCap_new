12 Month Rebate Amount Help

The 12 Month Rebate Amount screen allows the user to view a list of all past year's 12 Month Rebate Amount for Political Rebates.

The following fields can be viewed: Rebate Year, Rebate Percentage, Full-Time, Part-Time, Lower Part-Time, and Retiree.

Available actions on this screen are based on the user's privileges. Key actions available on this page include:

Edit (link) The system provides the user with the ability to Edit the 12 Month Rebate Amount Data of the previous rebate year.
Add Year (button) The system allows the user to Add the 12 Month Rebate Amount Data of the current rebate year.
Return (button) The system returns the user to the Enterprise Application Main Menu.
