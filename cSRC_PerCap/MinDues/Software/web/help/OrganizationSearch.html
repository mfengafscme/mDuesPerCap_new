Organization Search Help



This screen provides the user with the ability to search for Organizations.

This is an "editable" screen with two sections. The first section allows entry for general Organization information such as the Organization Name, Organization Type, the Organization Website and Marked for Deletion.

The next section contains fields associated with an Organization's Address and Phone Numbers. Address fields include Attention (to), Address 1, Address 2, City, State, Zip/Postal Code, County, Province, Country, Last Updated and a box for entering a userid for Updated By.

Phone Numbers allows the user to enter Office and FAX numbers in the Country Code, Area Code and Number fields.

To perform a search, at least one field must contain data.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Reset (Button) The system clears any values entered in the search fields.
Submit (Button) The system displays the Organization Search Results Screen with a list of results that match the search criteria.
