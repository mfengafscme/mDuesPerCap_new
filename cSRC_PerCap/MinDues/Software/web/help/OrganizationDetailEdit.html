Organization Detail Edit Help



This screen provides the user the ability to edit an Organization’s information.

This screen allows the user to edit the Organization Name, Organization Type, Organization Website and also to set whether this Organization should be Marked for Deletion.

The rest of the fields are for Location Information and are view only.

All required fields must be entered.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (Button) The system validates and saves Organization Detail changes.
Reset (Button) The system clears any changes made to the screen returning it to its original content.
Cancel (Button) The system returns to Organization Detail Screen with no changes.
