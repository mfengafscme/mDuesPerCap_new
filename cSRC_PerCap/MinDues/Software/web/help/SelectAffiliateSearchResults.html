Select Affiliate Help



This screen allows the user to select for an Affiliate for access to that Affiliate’s Data Utility Main Menu screen.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Previous (link) The system displays the previous group of results. Sort order is maintained.
Next (link) The system displays the next group of results. Sort order is maintained.
Sort (links) Sort fields are: Name, Type, Local/Sub Chapter, State, Sub Code, Council/Retiree Chapter. The system displays the results sorted by the field selected. If the results were already sorted by the field selected, they will be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field selected.
Select (links) The system accesses to the selected Affiliate’s Data Utility Main Menu screen.
New Search (button) The system displays Affiliate Search screen to allow the user to search for another Affiliate.
