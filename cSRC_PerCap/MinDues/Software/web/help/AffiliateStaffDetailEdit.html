Affiliate Staff Detail Edit Help



The Affiliate Staff Detail Edit screen provides the user the ability to edit an Affiliate Staff Person's information.

Prefix The user selects a prefix based on valid code values in the database.
First Name The user provides the first name.
Middle Name The user provides the middle name.
Last Name The user provides the last name.
Suffix Name The user provides the suffix based on valid code values in database.
Nickname The user provides the nickname.
Alt Mail Name The user provides an alternate mailing name.
SSN The user provides a Social Security Number.
Valid SSN The user selects whether or not the Social Security Number is valid.
Staff Title The user provides a staff title.
POC For The user selects a Point of Contact For based on valid code values in database.
Affiliate Employer Name The user provides the Affiliate Employer Name.
Enter New Comments The user provides comments.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.

Submit (button) The system validates and saves Staff Person changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to Affiliate Staff Detail Screen with no changes.
