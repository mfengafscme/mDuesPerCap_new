Selection Criteria Fields Help



This screen provides the user the ability to choose data fields for selection criteria.

There are four sections, Person, Member, Officer and Affiliate, on this screen with each containing a pre-determined list of fields. The system provides composite fields of "Full Name", "Full Address", and "Full Phone Number" in Person fields and the system provides composite fields for "Affiliate Identifier", "Primary Location Address" and "Primary Location Phone Number" in the Affiliate fields.

Clicking the heading of each sub section will select/de-select all the fields within that heading.

For Affiliate fields, only the Primary Location is searched for Address and Phone. Additionally, the Created User ID/Date and Last Updated User ID/Date listed under Affiliate Detail searches only for core Affiliate Detail screen modifications which does not include associated affiliate information.

The screen retains the user's selection information within a current user session.

If the user is editing a previously saved report, the data fields on the screen are pre-populated with the previously saved data.

The user does not need to choose any field on this screen to proceed to the next step.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Output Fields (Link) The system provides the user the ability to go back and select output fields for a query via this action.
Output Fields Order (Link) The system provides the user the ability to define placement order for the selected output fields in the output.
Selection Criteria Fields (Link) This is the current screen. It is disabled.
Selection Criteria (Link) The system provides the user the ability to define selection criteria for a query via this action.
Sort Criteria Fields (Link) The system provides the user the ability to select data fields for specifying sort criteria for a query.
Sort Criteria (Link) The system provides the user the ability to define sort criteria for a query via this action.
Preview Query (Link) The system provides the user the ability to preview, save, or cancel a query via this action.
Cancel (Button) The system provides the user the ability to cancel the process of maintaining a query and return the user to the entry screen of "Maintain Queries".
