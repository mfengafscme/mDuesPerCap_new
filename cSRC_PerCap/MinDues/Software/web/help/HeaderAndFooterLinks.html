Almost every page in the application has the same Header and Footer links. These links provide access to common application content or functionality. The Header links are located near the top of each screen, and the Footer links are located near the bottom of each screen.

Some screens will not have all of these links, based on the entry point and type of functionality being displayed.

The Header links are:
Main Menu Returns the user to the AFSCME Main Menu or Affiliate View Data Utility Main Menu
Need Help with this page? Displays the help page for that screen


The Footer links are:
LOGOUT Logs the user out of the AFSCME Enterprise Application and returns the user to the Welcome screen
AFSCME A link to www.afscme.org, AFSCME's web site
PRIVACY POLICY Displays a screen with information on AFSCME's privacy policy
HELP Provides the ability to download the End User Documentation for the AFSCME Enterprise Application system
