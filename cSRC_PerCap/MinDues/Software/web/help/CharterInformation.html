Charter Information Help



The Charter Information screen provides the user with the ability to view an Affiliate's Charter information.

The following Actions (links or buttons) may be available from this screen. Remember that the actions that are available will depend on the the user's privileges.
Disassociate (link) The system removes the Affiliate's Association with the Council and any Administrative/Legislative Councils above it in the hierarchy. This link is only viewable if there is a Council Affiliation currently set.
Associate (link) The system allows the user to choose a new Council. This link is only viewable if there is not a Council Affiliation currently set.
