Affiliate Identifier Finder Results Help



The Affiliate Identifier Finder Results screen will displays the results of an Affiliate Finder search contained on the originating screen. It gives the user the ability to sort the results or select an individual result to be used to populate the associated fields on the screen in which the finder originated.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.

Select (links) The system fills the Affiliate Identifier of the chosen Affiliate to it's corresponding fields on the parent screen and closes the Affiliate Identifier Finder Results window.
Previous (link) The system displays the previous group of results. Sort order is maintained. Active only if results are greater than 25 and the first group of results is not being viewed.
Next (link) The system displays the next group of results. Sort order is maintained. Active only if results are greater than 25 and the last group of results is not being viewed.
Sort (links) The system displays the results sorted by the field the user selected (Name, Affiliate Type, Local, State/National Type, Sub Unit, or Council). If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
