Participation Group Add Type Help



The Participation Group Add Type screen provides the user with the ability to add a new Participation Type code to an existing Participation Group.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (button) The system performs validations and saves the Type.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns the user to the Participation Group View Screen without saving the new Type.
