Participation Group View Help



The system provides the user the ability to view Participation Group.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Return (button) The system returns to the user to the Participation Group Maintenance screen.
Save As (button) Provides the user with the Export Participation Codes screen.
Add a Detail (button) Provides the user with the Participation Group Add Detail screen.
Add a Type (button) Provides the user with the Participation Group Add Type screen.
Note: If a Detail previously has been added to this Group, the Deactivate (button), Save As (button) and Add An Outcome (button) also appear on the Participation Group View screen.
Deactivate (button) allows the user to Deactivate this detail. Once deactivated, the detail will no longer be used or maintained. OK (button) deactivates this detail. Cancel (button) returns the user to the Participation Group View screen.
Save As (button) Provides the user with the Export Participation Codes screen.
Add An Outcome (button) Provides the user with the Participation Group Add Outcome screen.
