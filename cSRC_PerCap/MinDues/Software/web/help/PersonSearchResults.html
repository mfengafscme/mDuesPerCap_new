Person Search Results Help



This screen displays the results of a Person Search. Each row in the results represents a person that matched the Person Search Criteria. Basic information is displayed in the result list for each Person record represented in the result.

This screen supports paging, for result sets that are too large to conveniently fit on one screen. If the result set is too large to fit on one screen, additional information and links will be displayed in order to identify which rows are being displayed and to allow the user to page forward and backwards in the result set.

Some of the fields in the result set will not contain data, if the data does not exist for that person. For example, if the person is not a member of an affiliate, affiliate identifier fields will not be displayed. If the Enterprise Application system does not have an Address or Social Security Number(SSN) for that person, Address or SSN will not be displayed for that person in the result set.

The initial Sort sequence for records in the result set is (first to last):
  • Name(Last Name then First Name), ascending alphabetically
  • State/Nat'l, ascending alphabetically
  • CN/Ret Chap, ascending numerically
  • Local/Sub Chap, ascending numerically
The screen displays the following Fields for the rows in the result set:
Select Column contains the view link if the user has the necessary privilege
Name Concatenation of the person's name fields
Affiliate Identifier  
Type The affiliate type
Local/Sub Chap The local of a regular affiliate or Sub Chapter of a Retiree Chapter
State/Nat'l The two character abbreviation of the state of the affiliate or the national indicator
Sub Unit The sub unit designator of the affiliate, if the affiliate is a sub unit
CN/Ret Chap The council of a regular affiliate or the Retiree Chapter designation of a Retiree Chapter
Address Street address information
City The city of the address
State The two digit state abbreviation
Zip/Postal Code The United States zip code or Postal Code for a foreign country
SSN The United States Social Security Number for that person, if available
Userid The userid of the person in the AFSCME Enterprise Application


The screen also displays the text "Total Number of Matches:" and a value representing the number of records that matched the search criteria entered by the user. The text and value are displayed after the search results are displayed. This will be towards the bottom of the screen.

The following Actions (links or buttons) may be available from this screen. The actions available to the user will depend on the privileges that the user has and whether the result set contains enough records for paging to be enabled.

Previous (link) If paging is enabled, then selecting this link will display the previous set of results. The link will not be active if the screen is displaying the first set of records in the results.
Next (link) If paging is enabled, then selecting this link will display the next set of results. This link will not be active if the screen is displaying the last set of records in the results.
Add New Person (button) Displays the Verify Person screen.
New Search (button) Displays the Basic Person Search screen.
View (link) (By Row) Clicking on this link takes the user to the Person Detail screen for the selected person.


This screen contains column oriented actions for sorting the result set. Certain column headers display as links. Clicking on the link will sort the entire result set on that column. Clicking on the column header the first time will sort the results in ascending order based on the column, clicking on the same column again sort the results in descending order. When sorted in ascending order, a (+) plus sign will appear by the column. When sorted in descending order, a (-) minus sign will appear.
