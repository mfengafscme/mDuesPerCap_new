Email Address Maintenance - Edit Help



The Email Address Maintenance - Edit screen provides the user with the ability to edit email addresses associated with an individual.

The screen displays both primary and alternate email addresses (if known).

The user can designate an email address as "primary" to indicate this is the first choice for use, or "bad" to indicate that the email address is incorrect.

The user can key in email address information in both the Primary and Alternate fields.

The system populates the Last Updated and Updated By fields with the current system date/time and user ID, respectively, for any email address that was edited. If only one Email Address is entered, then it must be marked as Primary. If both are entered, then one of them must be marked as Primary.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Email Address Maintenance Screen.
Submit (button) The system performs validations and saves the changes.
