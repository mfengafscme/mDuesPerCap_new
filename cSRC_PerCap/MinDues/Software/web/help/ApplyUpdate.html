Apply Update Help



The Apply Update screen provides the user with the ability to view a list of the Member, Officer, and Rebate Update files that have been preprocessed for updating, are pending completion, processed, rejected update file reports, and update files that have been completed.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

Search Log (button) The system provides the user with the Apply Update Process Log screen.
Review (button) The system provides the user with the Apply Update Member Edit Summary screen.
Processed (button) The system allows the user to View Review Preprocessing Report and provides the user with the Apply Update Member Review Summary screen.
View (button) The system will submit the report for processing and the user will be provided with the Report Submitted screen. The Update has been scheduled to run for the selected Affiliates. An email will be sent once the Update has been applied to the system. The email provides the user a summary report showing the impact of applying the associated update file.
