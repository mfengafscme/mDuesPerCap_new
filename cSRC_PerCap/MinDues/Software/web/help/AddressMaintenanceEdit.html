Address Maintenance Edit Help



The Address Maintenance - Edit screen provides the user with the ability to edit an address associated with an individual.

An address can be designated as a primary address by putting a check in the box marked "primary".

In addition, using the appropriate check boxes, the address can be marked "bad" to tell the system not to send mail to that address and/or "private" to indicate that this address is restricted and only available to the department who entered it.

The fields that can be edited are Mailing Code (pull-down), Address 1 and Address 2, City, State (pull-down), zip code (including plus 4), County, Province (if Canada is selected as the country) and Country (pull-down).

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

Fields maintained by the system are Date Created, Created By, Last Updated and Updated By and cannot be edited.

NOTE: All required fields (marked with an asterisk) must be entered.

  • If the Country entered is Canada, the Province is also required.
  • It is required to have at least one Primary Address.
  • Only an address with a Code equal to 'Home' can be flagged as 'Primary'.
  • If an Address is flagged as 'Primary', it overrides the Address that was previously flagged as 'Primary'.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Address Maintenance Screen.
Submit (button) The system performs validations and saves the changes.
