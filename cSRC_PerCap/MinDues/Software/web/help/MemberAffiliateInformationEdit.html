Member Affiliate Information Edit Help



The Member Affiliate Information Edit screen provides the user with the ability to edit data specific to a member of a specific affiliate. If a member is associated with multiple affiliates, each affiliate may have different data for the member. Member Affiliate Information data includes:
  • Member Type
  • Member Status
  • Affiliate Member Number
  • Join Date
  • Retired Date
  • Lost Time Language in Contract
  • Dues Type
  • Dues Rate
  • Dues Frequency


The following Actions (links or buttons) are available from this screen depending on the privileges the user has been given.

Submit (button) The system processes the update and, if there are no errors, returns the user to the Member Affiliate Information Screen
Reset (button) The system resets the data displayed on the screen to the data when the screen was originally entered
Cancel (button) The system cancels the edit operation and returns the user to the Member Affiliate Information Screen
