Replace Officer Results Help



The Replace Officer Results screen allows a user to view all the possible candidates for a given office. Each column in the table has multiple headers that when clicked re-sort the information by the data under that header. The user can browse the results in screens of 25.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Select Closes the results window, and populates the name fields on the Edit Officer Maintenance screen with the name of the person chosen.
