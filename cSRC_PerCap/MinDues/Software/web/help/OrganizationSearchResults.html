Organization Search Results Help



This screen provides the user with the ability to view the results of an Organization Search and to navigate the results, sort the results, view the details of an individual result, or initiate a new search.

This is a three column screen. The 'Select' column allows the user to select "View" an Organization to see detailed information of an Organization. The remaining columns are: Organization Name and Organization Type.

Initial sort order will be by Organization Name ascending alphabetically.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

View (Link) The system displays detailed information about the selected Organization.
Previous (Link) The system displays the previous group of results. Sort order is maintained.
Next (Link) The system displays the next group of results. Sort order is maintained.
Sort By Field (Link) The system displays the results sorted by the field selected. If the results were already sorted by that field, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon that field.
New Search (Button) The system displays Organization Search Screen.
