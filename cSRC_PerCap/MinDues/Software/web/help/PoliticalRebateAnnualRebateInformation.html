Political Rebate - Annual Rebate Information Help



This screen provides the user the ability to view the final processing information associated with an individual’s Political Rebate.

The following Actions (links) may be available from this screen. Remember that the actions that are available will depend on the user's privileges.

Return (Button) The system returns the user to the Political Rebate Summary Screen.
Edit (Button) The system allows the user to edit the Political Rebate Information.
Clear (Link) The system removes the Affiliate from the Annual Rebate. (Once a check is issued, this option is no longer available)


When the user selects the ‘Remove an Affiliate’ action, the system will prompt the user to confirm the action.
