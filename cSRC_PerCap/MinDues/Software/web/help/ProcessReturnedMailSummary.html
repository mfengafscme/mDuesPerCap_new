Process Returned Mail Summary Help



The Process Returned Mail Summary screen provides the user with the ability to view a summary of the number of bad addresses flagged and not flagged. The screen also shows a list of addresses updated in the last month, and lets the user decide whether they should or should not be flagged.

To perform any action the user must have the required privilege set by the Admin functionality.



The following Actions (links or buttons) may be available from this screen.

Edit (button) The system allows the user to edit any address in the summary.
Done (button) The system returns the user to the main menu.
