Political Rebate - Summary By Year Help



The Political Rebate - Summary By Year screen lets the user view a list of all the years in which an individual has applied for a Rebate.

This is a "view only" screen, that shows the year for which a rebate was asked for and the status of that request. The values shown for status are: Denied, In Progress and Check Issued.

NOTE: The status field is calculated based on the status of the all the Requests and Applications, the Roster Status, the Acceptance Code of each Affiliate, and the Check Number, if at least one of the Affiliates approved the rebate.

The following Actions (links) may be available from this screen. Remember that the actions that are available based on the privileges given to the user.
View (Link) The system displays the summary information for the chosen rebate year.
Add (Link) The system displays the Political Rebate-Request Add Window that allows the user to add a new Rebate Year via adding a Request for that year.
