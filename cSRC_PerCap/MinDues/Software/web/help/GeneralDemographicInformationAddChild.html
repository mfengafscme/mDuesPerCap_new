General Demographic Information - Add Child Help



This screen provides the user with the ability to add new child information. Additional children can be added by using the fields provided. Birthdate can either be entered or selected with the "date picker" button.

NOTE: When entering new Child information, First and Last Names are required.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to General Demographic Information Screen with no changes saved.
Submit (button) The system validates and saves the new "Add Child" information changes.
