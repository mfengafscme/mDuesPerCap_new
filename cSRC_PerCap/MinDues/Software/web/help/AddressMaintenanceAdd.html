Address Maintenance Add Help



The Address Maintenance - Add screen provides the user with the ability to add a new address to an individual. Editable fields include address line 1, address line 2, city, state (using a pull-down), zip code, county, province and country. NOTE: Required fields are identified with an asterisk.

Using the checkboxes provided, an address can be marked as "bad" to tell the system not to send mail to that address; or "private" to indicate that this address restricted and only available to the department that created it. A checkbox is also used to designate an address as the "primary" address.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Address Maintenance Screen.
Submit (button) The system performs validations and saves the new address.


NOTE: When the data is submitted, the system will perform a validation process for the State entered against the zip code. If the validation fails, an error message will be generated.
