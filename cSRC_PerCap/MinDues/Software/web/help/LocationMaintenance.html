Location Maintenance Help



The Location Maintenance screen allows the user to view basic information pertaining to locations associated with an organization. A location may have both a regular and a ship to address. Each location may have a both an Office and Fax phone number associated with it. An organization may have multiple locations, one of which can be marked as the primary location for that organization.

Available actions on this screen are based on the user's privileges. Key actions available on this page include:
Edit (link) Edit an existing location. This link appears beside each editable location.
Delete (link) Delete an existing location. This link appears beside each deletable location.
Add (button) Displays the Location Maintenance Add screen which can be used to add a new location.
Return (button) The system returns to the screen that started the flow.
