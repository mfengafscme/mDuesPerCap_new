Person Detail Help



The Person Detail screen displays basic data about a Person in the AFSCME Enterprise Application system. This screen is primarily for viewing Person data., The Person Detail - Edit screen provides the ability to update basic Person data, providing the user has the privilege to access the Person Detail - Edit screen.

The screen contains Person information such as Name, Social Security Number (SSN), System Mailing Address (SMA) and Comments. Additional Person record related data can be found on other pages, such as General Demographics Information, which contains demographic information about the Person.

This screen also contains Actions (links) for accessing other Person related screens, such as Address Maintenance, Phone Maintenance, and Email Maintenance. These actions will only display if the user has the appropriate privileges.

There are several important aspects of this screen to highlight. First, the Person Tab will display at the top and bottom of the page. The tab consists of rectangles with embedded links in them. The links appear as words with underscores, for example, Search Results. If the link is inactive, it will not have an underscore, and will not be selectable. Each link will display a different screen. The number of links that appear is dependent on the privileges held by the user.More information on the Person Tab is provided below.

Second, there is a section near the top of the page that displays the Persona(s) of the Person. Personas are indicators of different data stored in the system for the Person. For example, the Person may be a Member of some Affiliate or belong to the an Affiliate's Staff. The Persona(s) will display only if the user has the appropriate privilege. The Persona(s) display as links which when selected, display the appropriate screen for that persona. For example, the Member persona link will display the Member Detail page.

Third, if a System Mailing Address (SMA) exists for the Person, the screen will display the System Mailing Address. It is possible that the Person does not have a SMA but may have other addresses. These addresses can be seen by selecting the Maintain Addresses link, if the user has the required privilege. All the phone numbers for the Person that are viewable by the user will be displayed. Select Maintain Phone Numbers link to navigate to additional pages for maintaining a Person's phone number. The email addresses for the user are all displayed on the Person Detail screen. Email addresses for the Person can be maintained by selecting the Maintain Email Addresses link. As usual, the user must have the appropriate privilege(s) for these actions to be available on the screen.

Fourth, if one exists, the last Comment associated with the Person will be displayed on the page. To access additional comments for the Person, the user can select the View More Comments link, if the user has the appropriate privilege.

The following Actions (links or buttons) may be available from this screen. The actions available to the user will depend on the privileges that the user has.
Member (under Persona) (link) Displays the Member Detail screen, if the Person is a Member
Affiliate Staff (under Persona) (link) Displays the Person Staff List screen, if the Person is an Affiliate Staff
Organization Associate (under Persona) (link) Displays the Person Organization Associate List screen, if the Person is an organization associate
Maintain Addresses (link) Displays the Address Maintenance screen
Maintain Phone Numbers (link) Displays the Phone Numbers Maintenance screen
Maintain Email Addresses (link) Displays the Email Address Maintenance screen
View More Comments (link) Displays the Comment History screen
