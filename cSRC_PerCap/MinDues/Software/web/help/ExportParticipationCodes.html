Export Participation Codes Help



The system provides the user the ability to select a file format for saving Participation Codes.

The user can select a radio button for one of the following Participation Code Export formats: Tab Delimited, Comma Delimited, or Semicolon Delimited.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system allows the browser to download the file in a separate window and returns the user to the Participation Group View Screen.
Cancel (button) The system returns the user to the Participation Group View Screen without saving the export file.
