Phone Number Maintenance Help



The Phone Number Maintenance screen provides the user with the ability to view or designate and/or delete a phone number associated with an individual.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Edit (link) This link (next to each phone number) will display the Phone Number Maintenance - Edit screen for editing phone numbers
Delete (link) The "Delete" link (next to each phone number) will cause the system to delete the phone number. Clicking on this link will cause a pop-up to appear that to confirm the removal of the phone number from the system.
Return (button) The system returns to the detail screen from which the Phone Number Maintenance Functionality was initiated.
Add (button) The system displays screen to allow the user to add a new phone number
