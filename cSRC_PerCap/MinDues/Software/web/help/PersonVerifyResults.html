Person Verify Results Help



The Person Verify Results screen displays information on similar persons who might be the same person as the one the user is trying to add, based on the criteria entered in the Verify Person screen. The user can view the results to determine if the person trying to be added matches someone already in the database. If not, the user can continue to add the new person.

This screen is used by a number of Add paths including:
  • Add Person
  • Add Member
  • Add Affiliate Staff
  • Add Organization Associate
If the user recognizes an existing person as the person trying to be added, the user can choose to add a Member or Affiliate Staff using that person or select a person to view more information.

This screen will only display if there are Person records in the AFSCME Enterprise Application Database which match the search criteria entered on the Verify Person screen. If no records match, the system will display a Detail - Add screen (which screen depends on the type of Add attempted) directly, without first displaying the Person Verify Results screen.

The initial Sort sequence for records in the result set is (first to last):
  • Name(Last Name then First Name), ascending alphabetically
  • Affiliate State ascending alphabetically
  • Affiliate Council ascending numerically
  • Affiliate Local ascending numerically
The following Actions (links or buttons) may be available from this screen. The actions available to the user will depend on the privileges that the user has.
Previous (link) If paging is enabled, then selecting this link will display the previous set of results. The link will not be active if the screen is displaying the first set of records in the results.
Next (link) If paging is enabled, then selecting this link will display the next set of results. This link will not be active if the screen is displaying the last set of records in the results.
Add New (button) Displays the Person Detail - Add screen, so the user can continue to add the new person.
New Search (button) Displays the Verify Person screen, so that the user can enter a new search for duplicate persons.
View (link) (By Row) Displayed by each row in the result set, if the user has the appropriate privilege to access the relevant Detail screen (e.g., Person Detail, Member Detail). Upon selection, displays the appropriate Detail screen for the current Add flow (e.g., Add Person, Add Member, Add Affiliate Staff)
Add (link) (By Row) Displayed by each row in the result set, if the user has the appropriate privilege. Upon selection, displays either the Member Detail - Add Affiliation or the Affiliate Staff Detail - Add screen. Not applicable in the Add Person flow.


This screen contains column oriented actions for sorting the result set. Certain column headers display as links. Clicking on the link will sort the entire result set on that column. Clicking on the column header the first time will sort the results in ascending order based on the column, clicking on the same column again sort the results in descending order. When sorted in ascending order, a (+) plus sign will appear by the column. When sorted in descending order, a (-) minus sign will appear.
