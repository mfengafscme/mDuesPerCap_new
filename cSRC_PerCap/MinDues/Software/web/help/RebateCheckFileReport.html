Political Rebate Check File Report Help



This screen gathers requested information to generate the Rebate Check File, Check Register, and Final Roster reports.

These reports provide a listing of the individuals that are getting checks, a listing of the members that have received the Rebate checks, and the final listing of the individuals that were given a Rebate for the current rebate year. These reports will be sent as needed to Accounting manually for processing.

The following steps are used to prepare for the Rebate Check File:

  • The Affiliate Relations Staff runs the report for the Rebate Check File and provides the starting check number.
  • The Enterprise System compiles a list of Members that have been approved for the Annual Rebate. If a single Affiliate approves a Rebate, then that Rebate is check eligible.
  • The Enterprise System verifies that a check amount has been calculated for each Member.
  • The Enterprise System assigns a check number for each Requestor and sets the check date.
  • The Check File is created and available for download.
  • The Check Register report is available for viewing and printing.
  • Requesters/Members are moved to the Final Roster and Political Rebate reports are updated accordingly.

The following Actions (buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Submit The system generates the Rebate Check File, Check Register, and Final Roster reports.
