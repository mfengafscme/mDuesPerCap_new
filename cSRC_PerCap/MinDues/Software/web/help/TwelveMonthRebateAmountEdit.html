12 Month Rebate Amount Edit Help



The 12 Month Rebate Amount Edit screen allows the user to edit Rebate Amount information for the most recent existing year.

The Rebate Percentage (text box), Full-Time (text box), Part-Time (text box), Lower Part-Time (text box), and Retiree (text box) are required fields.

Available actions on this screen are based on the user's privileges. Key actions available on this page include:

Submit (button) The system performs validations and saves the changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the 12 Month Rebate Amount screen.
