Organization Detail Add Help



This screen provides the user with the ability to add a new Organization.

This is an "editable" screen that collects basic information about the Organization including Organization Name, Organization Type and Organization Website.

All required fields must be entered.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (Button) The system validates and saves new Organization data.
Reset (Button) The system clears any changes made to the screen returning it to its original content.
Cancel (Button) The system returns to the AFSCME Main Menu
