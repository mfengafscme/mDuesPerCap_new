Person Staff List Help



This screen provides the user the ability to view a list of all the Affiliate Staff Positions a Person holds.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

View (links) The system displays the selected Staff Person’s Detail Information.
Sort (links) Sort fields are: Affiliate Type, Local, State/National Type, Sub Unit, Council, Affiliate Staff Title, Point of Contact For, and Location.
The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
