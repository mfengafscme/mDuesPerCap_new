Select Affiliate Search Help



The Select Affiliate Search screen provides the user with the ability to search for Affiliates in order to view the Data Utility Main Menu of the selected affiliate.

The following fields are available for the search:
  • Type (pull-down)
  • Local/Sub Chapter (text box)
  • State/National Type (pull-down)
  • Sub Unit (text box)
  • Council/Retiree Chapter (text box)
The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system displays the Select Affiliate screen with a list of results that match the search criteria.
Reset (button) The system clears any changes made to the screen and returning it to its original content.
Cancel (button) The system returns to the Enterprise Application Main Menu.
