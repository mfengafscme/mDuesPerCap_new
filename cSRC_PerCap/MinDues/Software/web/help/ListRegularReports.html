Generate Reports Main Menu Help



This screen lists the regular reports that the user can access. Basic information on the report(s) are provided such as Name, Description, Last Updated User Id and Last Updated Date. If the report name appears as a link (underlined), click on the report name to initiate processing for the report.

Note that the user will only see reports that they have been granted access to generate.

At the bottom of the screen is a section labeled My Reports. This section may contain saved queries from the Maintain Queries functionality that were created by the user.
