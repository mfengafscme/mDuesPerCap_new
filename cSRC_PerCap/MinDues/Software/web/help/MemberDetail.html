Member Detail Help



This screen provides the user with the ability to view an individual Member's data and perform specific actions related to the Member’s information.

This screen displays general information about a Member. Some of the same information is available on the Person Detail screens.

The Maintain Address section displays address information and provides a link to Add, Edit, or Delete an address for the Member.

The Member/Affiliate section is divided into these sections:

  • View provides links to allow the user to view the Member’s Affiliate Information and Affiliate Detail Information.
  • Affiliate Associations displays the affiliates that the Member belongs/belonged to and shows the following sub categories/columns: Type, Local/Sub Chap, State/Nat'l, Sub Unit and CN/Ret Chap.
  • Member shows the current Type and Status.
  • Officer Position Held displays the position and each is linked so the system provides a list of the associated Affiliate’s Officers.
  • Officer Mail To is a link to view the Mailing Address associated with an Officer Position. The Person’s Address Code (Home, Work, etc.) or the Affiliate’s Location Title based on where mail is set to go will display or ‘SUSPENDED’ if Officer is Suspended with no link, or ‘ADMINISTRATORSHIP’ if the Affiliate is under Restricted Administratorship with no link.

NOTE: Under the Affiliate View Data Utility(VDU) for the displayed Member, only their affiliations within the user’s current data hierarchy will be displayed. Any Member affiliation for that person outside of the Affiliate Hierarchy of the VDU User will not be displayed. The Affiliate Hierarchy is determined by the affiliate the user entered the Affiliate View Data Utility through.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Maintain Addresses (link) Displays the Address Maintenance screen
Member (link) Displays the Member Affiliate Information screen
Affiliate (link) Displays the Affiliate Detail screen
Add (link) Displays the Member Detail - Add (Affiliation) screen
Maintain Phone Numbers (link) Displays the Phone Numbers Maintenance screen
Maintain Email Addresses (link) Displays the Email Address Maintenance screen
View More Comments (link) Displays the Comment History screen
