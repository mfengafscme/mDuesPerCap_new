Code Type Help



The system allows the user to view existing codes and to add, delete, and edit codes.

Common codes are reference values used by the system. For example, state abbreviations and state descriptions are considered to be a common code. A common code type is the definition of a set of reference values, e.g. state (VA - Virginia, UT - Utah).

The following Actions (buttons) are available from this screen. The actions available will depend on the privileges each user has been given.

Click the existing Code name (button) The system allows the user to edit a Code.
Add Code Type (button) The system displays the screen that allows the user to add a new code.
Submit (button) The system performs validations and saves the changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Enterprise Application Main Menu Screen.
