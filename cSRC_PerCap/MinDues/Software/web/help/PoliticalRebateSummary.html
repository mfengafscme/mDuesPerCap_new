Political Rebate - Summary Help



The Political Rebate - Summary screen provides the user the ability to view summary information for a Rebate in a given year.

This screen provides information concerning Requests, Applications, and Annual Rebates. The Requests List shows the Date of Request, Application Mailed Date and the Affiliate Identifier where Dues Paid To. The Status window displays the status of the request.

The Applications section provides Application Mailed Date and the Affiliate Identifier where Dues Paid To. The Status window tells the user the status of the Application.

The Annual Rebate Information section provides the ability to Select (view) Rebate Information and observe the Status.



The following Actions (links) may be available from this screen. Remember that the actions that are available based on the privileges given.

View (Link) The system displays the Political Rebate Request screen.
Add (Link) Add a new Rebate Request received from the Person.
View (link) The system displays the Political Rebate Application screen.
View (link) The system displays the Political Rebate Annual Rebate Information screen.
