Power Affiliate Search Help



The Power Affiliate Search screen provides the user with the ability to search for Affiliates using fields from the Affiliate Detail screen.

To perform a search, at least one field must contain data.


The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality.

Submit (button) The system displays the Affiliate Search Results Screen with a list of results that match the search criteria.
Reset (button) The system clears any values entered in the search fields.
Power Search (button) The system displays advanced search functionality that allows the user to search on any field associated with the Affiliate's detail information.
