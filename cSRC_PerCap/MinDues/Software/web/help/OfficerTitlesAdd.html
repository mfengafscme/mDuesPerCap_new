Officer Titles Add Help



The Officer Titles Add screen provides the user with the ability to add a new Officer Title to an Affiliate. If Auto Delegate flag is set, at least one position must have a delegate priority set. The Auto Delegate flag is only editable if Approved Constitution is checked.

Fields with an asterisk * are required fields.

Constitutional Title (pull-down) The user is to select a constitutional title.
Affiliate Title (textbox) The user is to provide an affiliate title.
# with Title (textbox) The user is to provide number of offices for this title. Only one President is allowed. Once President position exists, it can no longer be added.
Month of Election (pull-down) The user is to select the month of election.
Length of Term (pull-down) The user is to select the length of term. Only the following positions are allowed to have a Length of Term equal to 'Indefinite':
  • Financial Reporting Officer
  • Administrator
  • Steward
  • Executive Director
  • Director
    		•
    Delegate Priority (numeric) The user is to provide delegate priority.
    RO (checkbox) The user is to select whether or not to this officer title is a reporting officer.
    E-Bd (checkbox) The user is to select the whether or not this officer title is an executive board member.
    The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

    Submit (button) The system validates and saves the new Officer Title.
    Reset (button) The system clears any changes made to the screen returning it to its original content.
    Cancel (button) The system returns to the new Officer Title.
