Member Detail Add Help



This screen provides the user with the ability to add a new Member which includes adding them as a new Person to the system.

All required fields must be entered.

The Affiliate Identifier must exist in the system.

The Affiliate must have a status of either Chartered, Not Chartered, Restricted Administratorship, or Unrestricted Administratorship to be valid.

The Member Type can only be ‘Retiree’ and ‘Retiree Spouse’ if Affiliate Type is ‘R’ or ‘S’.

If the Person being added has a value entered for Social Security Number, the system will check if that Social Security Number already exists. If it does already exist, the system will show the Duplicate SSN Notifier screen before performing the requested action, thus allowing the user to continue or cancel the action. If continuing the action, the records that match are flagged as duplicates.

If the Affiliate Finder functionality is not used to build the fields of the Affiliate Identifier and duplicate Affiliates are found for the data entered, the system will display the Duplicate Affiliate Identifier Notifier Screen as part of the flow to allow the user to indicate the correct Affiliate. Once the choice is made, the flow continues with the requested action.

If one of the Address fields is populated, the system will validate the following rules:

  • If Country is Canada, the Province is required.
  • If one of the Phone fields is populated, then all of the remaining Phone fields are required.


The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (Button) The system validates and saves new Member data.
Submit & Add Another (Button) The system validates and saves new Member data, and allows the user to enter another Member.
Reset (Button) The system clears any changes made to the screen returning it to its original content.
Cancel (Button) The system returns to the detail screen from which the Member Detail Functional Specification was initiated
