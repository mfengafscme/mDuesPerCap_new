Organization Detail Help



This screen provides the user with the ability to view an Organization's data and to perform actions on that Organization’s information.

The first section provides the Organization Name, Organization Type, Organization Website and whether Organization has been Marked for Deletion or not.

The Maintain Location Information section displays the Primary address, if one exists, and the Office and FAX Phone Numbers associated with the Primary address.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Maintain Location Information (link) The system allows the user to add, edit, and delete Location information for this Organization.
