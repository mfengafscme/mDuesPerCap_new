General Demographic Information - Edit Help



This screen provides the user with the ability to edit an individual's General Demographic Information as follows:
Birth Date Enter either the information in the appropriate field or use the "Date Picker" button to the right of the field. Using the "Date Picker" button causes a pop-up day/month/year calendar to display and allows the selection of the date.
Deceased Date Enter the information in the appropriate field or use the "Date Picker" button to the right of the field. Using the "Date Picker" button causes a pop-up day/month/year calendar to display and allows the selection of the date.
Gender Radio buttons for Male, Female and "Unknown" enable gender selection
Ethnic Origin A pull-down menu (based on a code table) allows selection of Ethnic Origin
Citizenship A pull-down menu (based on a code table) allows Status selection of Citizenship Status
Marital Status A pull-down menu (based on a code table) allows selection of Marital Status
Disabilities A multiple selection menu (based on a code table) allows input of specific disability condition(s)
Special Accommodation Requirements A multiple selection menu (based on a code table) allows input of special accommodation requirements.
Primary Language A pull-down menu (based on a code table) allows selection of a Primary Language
Other Languages A multiple selection menu (based on a code table) allows input of other languages
Religion A pull-down menu (based on a code table) allows selection of Religion

Entry fields are provided for entering relevant information under the Spouse/Domestic Partner and Children Information headers.

NOTE: When entering information in the Spouse/Domestic Partner Information fields, First and Last Names are required. In the Children Information Field, a First name is required. "Date Picker" buttons are available for Children birthdate entry.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to General Demographic Information Screen with no changes saved.
Submit (button) The system validates and saves General Demographic information changes.
