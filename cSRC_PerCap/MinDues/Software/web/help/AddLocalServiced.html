Affiliate Staff Add Local Serviced Help



The Affiliate Staff Add Local Serviced screen provides the user the ability to add a Local that is serviced by this Staff Person.

To perform any action, the user must have the required privilege, which is set by the Admin functionality.

The following Actions (links or buttons) may be available from this screen. All fields with an asterisk * are required fields.

Affiliate Identifier (link) Displays the Affiliate Finder to assist in retrieving the Affiliate Identifier.
Submit (button) The system validates and saves the new Local Serviced.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to Affiliate Staff Detail Screen without adding a new Local Serviced.
