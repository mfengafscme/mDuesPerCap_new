Officer History For Affiliate Help



The Officer History For Affiliate screen provides the user with the ability to view a listing of the current and past officer positions held for an Affiliate.

The following Actions (links or buttons) are available from this screen depending on the privileges you have which is set by the Admin functionality.

Sort (links) RO Flag, Officer Title (by precedence order), Concatenated Name, Beginning Term Date, Ending Term Date, and Member Number. Initial sort order will be by Beginning Term Date descending chronologically. The system redisplays the same screen with the list sorted by the field the user selected. If the list was already sorted by the field the user selected, it should be displayed in the reverse order. Otherwise, the list is displayed as sorted in ascending order based upon the field the user selected.
