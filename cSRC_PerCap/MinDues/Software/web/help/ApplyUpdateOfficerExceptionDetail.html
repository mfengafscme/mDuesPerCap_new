Apply Update Officer Exception Detail Help



The Apply Update Officer Exception Detail screen provides the user with the ability to view detail statistical information on an individual exception listed on either the Officer Edit Summary or the Officer Review Summary. This screen highlights the differences between the data received in the file and the data in the system for an individual.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Return (button) The system returns the user to the summary report from which this action was initiated.
