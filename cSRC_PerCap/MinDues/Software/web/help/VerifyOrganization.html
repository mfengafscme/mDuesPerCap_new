Verify Organization Help



This screen is used as a basic search tool to determine if an Organization record already exists before adding a new one. It is part of the Add flow for Organizations.

This screen allows the user to enter an Organization Name.

All required fields must be entered.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (button) The system displays the Organization Verify Results Screen with a list of results that match the search criteria.
Reset (button) The system clears any changes made to the screen returning it to its original content.
