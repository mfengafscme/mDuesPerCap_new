My Personal Information Help



The My Personal Information provides the user with the ability to view a subset of their own information.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Update My Personal Information (button)
Add (button) The system displays the screen that allows the user to add a new phone number.
Update My Personal Information (button) The system displays the screen that allows the user to add or edit a new address or email address.
