Mailing Lists Address Association Help



This screen provides the user with the ability to change the Address (for an individual) or the Location (for an Organization or an Affiliate) that is associated with a Mailing List.

For an individual address, only addresses that do not have the Bad Address flag set will be included.

For organization or affiliate locations, only address type Regular will be included.

The following Actions (buttons or links) are available from this screen depending on the user's privileges.
Select (Link) The system sets the chosen address as the Mailing Address for the Mailing List.
Cancel (Button) The system leaves the currently selected address as the Mailing Address for the Mailing List.
