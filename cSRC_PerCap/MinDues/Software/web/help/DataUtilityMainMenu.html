Data Utility Main Menu Help



The system provides the user the ability to initiate the actions available to an Affiliate user.

The following Actions (links) are available from this screen depending on the user's privileges.

 
Search
Members (link) The system allows the user to Search for Members within her/his Affiliate and its Sub-Affiliates.
Add New Member (link) The system allows the user to Add a new Member within her/his Affiliate and its Sub-Affiliates.
Officers (link) The system allows the user to Search for Officers within her/his Affiliate and its Sub-Affiliates.
Affiliates (link) The system allows the user access to her/his Affiliate and its Sub-Affiliates.
 
Reports
Generate Reports (link) The system gives the user the ability to generate reports for her/his Affiliate and its Sub-Affiliates.
Maintain Queries (link) The system gives the user ability to add, delete and edit ad-hoc queries for her/his Affiliate and its Sub-Affiliates.
Generate Mailing Lists (link) The system gives the user the ability to generate mailing lists for her/his Affiliate and its Sub-Affiliates.
Download Latest Update File with Enriched Data (link) The system gives the user the ability to download a version of the Member Update file for her/his Affiliate with Enriched Data.
 
Import/Export
Upload Affiliate Member Data (link) The system gives the user the ability to upload the Member Update file for her/his Affiliate.
Upload Affiliate Officer Data (link) The system gives the user the ability to upload the Officer Update file for her/his Affiliate.
Upload Affiliate Rebate Data (link) The system gives the user the ability to upload the Rebate Update file for her/his Affiliate.
 
Maintenance
View Apply Update (link) The system gives the user the ability to view Apply Update Queue, which contains the pre/post-processing reports for her/his Affiliate created by the Apply Update functionality.
 
Administration
Member Privileges (link) The system allows the user to set the access to Personal Data for the Members in her/his Affiliate and its Sub-Affiliates.
AFSCME Main Menu (link) The system returns the user to the AFSCME Main Menu screen.
Switch Affiliate (link) The system allows the user to search for an new Affiliate in order to view that Affiliate’s Data Utility Main Menu Screen.
 
My Account
View My Personal Information (link) The system allows the user to view and edit their Personal Information.
Change My Password (link) The system allows the user to edit her/his own Password and Secret Question/Answer.
