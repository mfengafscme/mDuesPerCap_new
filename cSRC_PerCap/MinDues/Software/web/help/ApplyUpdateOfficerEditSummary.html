Apply Update Officer Edit Summary Help



The Apply Update Officer Edit Summary screen provides the user with the ability to view pre-processing statistical information on the Officer update and give the user the ability to apply or reject the update.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Perform Update (button) The system completes the requested update for the chosen Affiliates.
Cancel (button) The system returns the user to the Apply Update screen.
Reject (button) The system aborts the requested update and prompts the user for comments as to why they are canceling.
View (link) The system displays the Exception Detail screen.
