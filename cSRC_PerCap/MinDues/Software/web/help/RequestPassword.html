Request Password Help



This screen allows the user to request for the password to be sent to the e-mail address that was registered with the user ID. The Security Question that is filed in the User Account information record will be displayed. The user will have to response for the appropriate Security Answer in order to proceed with the request.

The following Actions (buttons) are available from this screen depending on the user's privileges.
Submit (button) The system performs validation on the supplied information. If validation succeeds, the password for the user ID will be sent to the e-mail that was supplied with the User Account information. If validation fails, this screen will be re-displayed along with a list of reasons validation failed. Correct the errors and click Submit button again.
