Code Type Edit Help



The system allows the user to edit, delete or add a code Type.

The user can edit:
Name (text box)  
Description (text box)  
Category (pull-down)  


The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Add Code (button) Provides the user with the Add Code screen.
Submit (button) Commits the changes made to this screen.
Delete (button) The system provides the user the opportunity to delete the existing Code.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Code Type Screen.
Sort (link) Each column header is a link that allows sorting by that column.
Edit (link) Provides the user with the Code Edit screen.
Delete (link) Provides the user the opportunity to delete the existing Code.
