Address Maintenance Help



The Address Maintenance screen provides the user with the ability to view or delete an address associated with an individual. Addresses are displayed by Department that created the address. A Department can only update and remove address associated with their Department.

On this screen the "primary" address is determined by the computer to be the "best" system mail address. It is the address that will be used to mail AFSCME material to this Person.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Edit (link) This link (next to each address) will cause the system to display the screen that allows the user to edit an address
Delete (link) The "Delete" link (next to each address) will cause a pop-up to appear that confirms the removal of the address.
Return (button) The system returns to the detail screen from which the Address Maintenance Functionality was initiated.
Add (button) The system displays the screen that allows the user to add a new address.
