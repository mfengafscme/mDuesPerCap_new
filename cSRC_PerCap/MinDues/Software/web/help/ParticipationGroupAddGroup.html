Participation Group Add Group Help



The system provides the user the ability to add a new Participation Group.

Name and Description are required fields.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (button) The system performs validations, saves the Group and provides the user with the Participation Group View screen.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns the user to the Participation Group Maintenance screen without saving the new group.
