General Demographic Information Help



This screen provides you with the ability to view an individual's General Demographic Information. This screen is used primarily for viewing however, the General Demographic - Edit screen (accessed with the "edit" button on the person tab) does provide the ability to edit information.

Also, additional children can be added by using the link to the General Demographic Information - Add Child screen, or deleted with the Delete Link.

NOTE: When you select the 'Delete an Existing Child' action via the link, the system will prompt you to confirm the action with the following message: "Are you sure you want to delete this Child's Information?"

Information on a spouse is added and edited through the General Demographic - Edit screen. Information on existing children can also be updated through the General Demographic - Edit screen.

This screen contains general demographic information such as Birth Date, Deceased Date, Gender, Ethnic Origin, Citizenship Status, and Marital Status. Also noted are: Disabilities such as Diabetes or Hearing loss. Special Accommodation Requirements are listed such as refrigeration for medication or the requirement for an Assisted Listening Device. Primary Language, Other Languages and Religion are also displayed.

In Release 1 of the AFSCME Enterprise Application, Spouse and Children are not recorded as Person records and therefore won't show in a Person search.

In Release 2 of the AFSCME Enterprise Application, the intention is to have information on Spouse and Children recorded as separate Person records. This screen displays the Person Tab at the top and bottom of the screen.

The following Actions (links or buttons) are available from this screen depending on the privileges you have which is set by the Admin functionality.

Delete (link) The system deletes the child information.
Add (link) The system will display the General Demographic Information Add Child screen to allow you to add new child information.
