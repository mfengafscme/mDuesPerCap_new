Verify Person Help



This screen is used as a basic search mechanism to determine if a Person record already exists before adding a new one and is part of the Add flow for all Person-type entities.

This screen allows the user to enter a First Name, Last Name, Suffix with a dropdown menu, a birth date with an optional calendar and Social Security Number.

All required fields must be entered.

Either 'First Name' and 'Last Name' or 'Social Security Number' is required.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

Submit (button) The system displays the Person Verify Results Screen with a list of results that match the search criteria. If no Person is found, Person Detail Add Screen will display.
Reset (button) The system clears any changes made to the screen returning it to its original content.
