Account Info Help



The Account Info screen provides the user with the ability to change their Password and Secret Question/Answer.

The Fields that can be edited are:
Change Current Password (text box)
Choose a new password (text box)
Re-Enter new password (text box)
Challenge Question (pull-down)
Challenge Response (text box)


The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality for the user.
Submit (button) The system performs validations and saves the modified account information.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Enterprise Application Main Menu Screen.
