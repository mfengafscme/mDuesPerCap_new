Basic Affiliate Search Help



The Basic Affiliate Search screen provides the user with the ability to search for Affiliates based on the most common search criteria. To perform a search, at least one field must contain data.

To perform a search, at least one field must contain data.

Type The user is able to provide an Affiliate type.
Local/SubChapter The user is able to provide a Local or SubChapter number.
State/National Type The user is able to select a state or national type.
Sub Unit The user is able to provide a Sub Local code.
Council/Retiree Chapter The user is able to provide a Council or Retiree Chapter number.


The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality.

Submit (button) The system displays the Affiliate Search Results Screen with a list of results that match the search criteria.
Reset (button) The system clears any values entered in the search fields.
Power Search (button) The system displays advanced search functionality that allows the user to search on any field associated with the Affiliate's detail information.
