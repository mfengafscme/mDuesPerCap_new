Person Detail Edit Help



The Person Detail Edit screen provides the user with the ability to edit a Person’s information if the user has the necessary administrative privileges. The "note" at the bottom of the screen identifies that any fields marked with an asterisk are required. On this screen the required fields are: First Name and Last Name. If data is not entered in the required fields, an error message will be generated and displayed.

Fields available to edit/update Person information include: First Name, Middle Name, Last Name, Suffix, Nickname, Social Security Number (SSN), Alternate Mail Name and Comments. This screen also allows the user to mark (using checkboxes) an address as bad, mark the Person record for deletion and mark the social security number as valid or a duplicate.

NOTE: If the Person whose details are being edited has a value entered for Social Security Number, the system will check if that Social Security Number already exists. If it does already exist, the system will show the Duplicate SSN Notifier screen before performing the requested action to allow the user to continue or cancel the action. By choosing to continue the action, the records that match will be flagged as duplicates.

The following Actions (links or buttons) are available from this screen depending on the privileges set by the Admin functionality for the user.

Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to the Person Detail screen without saving changes.
Submit (button) The system validates and saves changes. The system will update the "Last Updated" and "Updated By" fields with the appropriate data.
