Political Rebate - Request Help



This screen provides the user with the ability to view a Rebate Request that was submitted by an individual.

This is a two-section screen. The first section displays the following general Request information:
  • Rebate Year, Request Date, Certified Mail Number
  • Request Denied, Reason Denied and the Date Denied
  • Request Re-submitted, Re-submission Denied and Date Denied
  • Keyed Date
The next section displays the Affiliates from which the Rebate is being requested. The Select column contains links that will "clear" or remove the Affiliate from the list. Dues Paid To is the Affiliate Identifier information and has the following sub categories: Type, State/National, Local/Sub Chapter and Council/Retiree Chapter. The Duration identifies the time as a Member in the Affiliate during the Rebate Year and Filed With indicates where the request for a rebate was filed. Comment contains any notes from users maintaining this Request.

The following Actions (links) may be available from this screen. Remember that the actions that are available will depend on the user's privileges.

Edit (Button) The system allows editing the request and displays the Political Rebate Request Edit screen. Only available if the Request is still editable.
Return (Button) The system returns to the Political Rebate Summary By Year Screen. Once the Request has been given an Application Mailed Date value, it is no longer editable.
Clear (link next to each Affiliate Identifier) The system removes an Affiliate from the Request. This action is only available if the Request is still editable.
