Preview Query Help



This screen provides the user the ability to preview a query and the option to save or cancel this query.

Query Name cannot contain a space or start with strange character. Use an underscore ("_") instead of space.

If the user selects the "Mailing List" checkbox, the system enables the "Update Correspondence History" checkbox. If the user unselects the "Mailing List" checkbox, the system shall uncheck (if it is checked) and disable the "Update Correspondence History" checkbox.

If the user clicks "Save As" button, the system enforces the rule that the "Query Name" cannot be the same as any query that already exists.

The report section will dynamically be created based on the query created. The Output fields are displayed in, if defined, the order provided and the result set is limited based on any selection criteria identified and shown in the sort order defined.

If the user had selected "Count Query", the only output data field for the query is "Count". If the "Count Query" had been selected, all the data fields on the screen shall be disabled and the "Output Fields Order", "Sort Criteria Fields", and "Sort Criteria" links on the top of screen shall be disabled on all screens within the wizard chain for the Maintain Queries functionality.

The screen retains the user's selection information within a current user session.

If the user is editing a previously saved report, the data fields on the screen are pre-populated with the previously saved data.

Note: The "Mailing List" and "Update Correspondence History" fields will not be available if the user is an affiliate user.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Output Fields (Link) The system provides the user the ability to go back and select output fields for a query via this action.
Output Fields Order (Link) The system provides the user the ability to define placement order for the selected output fields in the output.
Selection Criteria Fields (Link) The system provides the user the ability to select data fields for specifying selection criteria for a query via this action.
Selection Criteria (Link) The system provides the user the ability to define selection criteria for a query via this action.
Sort Criteria Fields (Link) The system provides the user the ability to select data fields for specifying sort criteria for a query.
Sort Criteria (Link) The system provides the user the ability to define sort criteria for a query via this action.
Preview Query (Link) This is the current screen. It is disabled.
Save (Button) The system provides the user the ability to save a query persistently.
Save As (Button) The system provides the user the ability to reuse an existing query, and save it into a new query.
Cancel (Button) The system provides the user the ability to cancel the process of maintaining a query and return the user to the entry screen of "Maintain Queries".
