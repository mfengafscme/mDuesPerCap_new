Select Field Privileges Help



The system allows the user to view, add (check box) and edit (check box) field privileges for this role.

The following Actions (links or buttons) are available from this screen depending on the user's privileges.

Submit (button) The system performs validations and saves the Select Field Privileges.
Cancel (button) The system cancels the selection of fields returns to the Role Detail Screen.
Reset (button) Resets the form.


Note: Selecting a field name will allow the user to select or deselect all privileges in that field.
