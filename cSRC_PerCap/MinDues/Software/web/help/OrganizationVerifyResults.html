Organization Verify Results Help



This screen provides the user with the ability to view the results of the search entered in the Verify Organization screen and is part of the Add flow for Organizations.

This is a three column screen. The Select column allows the user to select "View" an Organization to see detailed information. The remaining columns are Organization Name and Organization Type. These fields have "links" that allow the system to display the results sorted by that column. If the results were already sorted by that column, the results should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon that column.

Initial sort order will be by Organization Name ascending alphabetically.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

View: (Link) The system displays detailed information about the selected Organization.
Previous (Link) The system displays the previous group of results. Sort order is maintained.
Next (Link) The system displays the next group of results. Sort order is maintained.
Add New (Button) The system displays the Organization Detail Add Screen allowing the user to enter a new Organization.
New Search (Button) The system displays Verify Organization Screen.
