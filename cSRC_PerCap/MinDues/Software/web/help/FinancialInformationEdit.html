Financial Information Edit Help



The Financial Information Edit screen provides the user with the ability to edit an Affiliate's Financial information.

Available actions on this screen are based on the user's privileges. Key actions available on this page include:
Fields with an asterisk * are required fields.

Submit (button) The system validates and saves Financial Information changes.
Reset (button) The system clears any changes made to the screen returning it to its original content.
Cancel (button) The system returns to Financial Information Screen.
