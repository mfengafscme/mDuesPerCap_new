Organization Associate Detail Add Help



This screen provides the user the ability to add a new Organization Associate to an Organization.

The following fields are not editable if the user chose the ‘Add from Existing Person’ action on the Person Verify Results Screen.
  • Prefix
  • First Name
  • Middle Name
  • Last Name
  • Suffix
  • Nickname
  • Alternate Mailing Name
  • Social Security Number
  • Valid Social Security Number
The following Actions (buttons) are available from this screen depending on the user's privileges set by the Admin functionality.
Submit The system validates and saves new Associate data.
Reset The system clears any changes made to the screen returning it to its original content.
Cancel The system returns to the Person Verify Results Screen.
