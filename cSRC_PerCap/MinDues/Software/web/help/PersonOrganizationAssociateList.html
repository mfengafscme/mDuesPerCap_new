Person Organization Associate List Help



This screen provides the user the ability to view a list of all the Organizations to which a Person belongs.

The following Actions (links or buttons) are available from this screen depending on the user's privileges set by the Admin functionality.

View (links) The system displays the selected Organization Associate's Detail Information.
Sort (links) Sort fields are: Organization Name, Organization Title, and Location. The system displays the results sorted by the field the user selected. If the results were already sorted by the field the user selected, they should be displayed in the reverse order. Otherwise, the results are displayed as sorted in ascending order based upon the field the user selected.
Return (button) The system returns to the Person Detail Screen.
